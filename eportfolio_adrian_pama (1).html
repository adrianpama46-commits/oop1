Midterm Seatwork 1 Topic
What are Operators?

Operators are symbols that instruct Java to perform a specific action on values. Arithmetic operators (+, -, *, /, %) handle math. Relational operators (==, !=, >, <, >=, <=) compare values and return true or false. Logical operators (&&, ||, !) combine conditions. Operator precedence — the order Java evaluates parts of an expression — determines whether the result is correct.

Reflection

Operators looked simple until the seatwork revealed the details that matter — integer division silently drops decimals, short-circuit evaluation can skip entire conditions, and a single missing parenthesis can change a result entirely. This topic made me more deliberate when writing any expression in Java, because the difference between "looks right" and "is correct" often comes down to one small decision.

Midterm Activities 5 Java Programs
JavaVariables & Data Types
Concept

A variable is a named storage location in memory that holds a value during a program's execution. In Java, every variable must be given a data type — int for whole numbers, double for decimals, String for text, boolean for true/false values, and so on. The type tells Java how much memory to reserve and what operations are valid on that variable.

ExplanationThis activity focused on analyzing a Java program involving variables, particularly how values are stored and changed in memory. It required tracing the program step-by-step, identifying how variables are assigned, updated, and referenced throughout execution. The task also involved completing tables and explaining the changes logically, helping visualize how data flows within the program. What I LearnedFrom this activity, I learned how variables behave in Java and how their values change during execution. It improved my understanding of memory tracing, assignment, and variable references. I also developed better analytical skills by carefully following each step of the program, which helped me become more accurate in understanding how code works behind the scenes.
